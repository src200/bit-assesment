## React Component for rendering text

A basic div that renders some text

### Component usage
```js
<Content>
    <h1>Hello World</h1>
    <p>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed euismod
        nisi euismod, porttitor nisl eu, tincidunt nisl. Nullam euismod
        dignissim nisi, eu aliquam eros.
    </p>
</Content>
```
